Only the distillation is model-authored: each section's summary, the glossary, the orientation context, and the suggested questions; everything else in the tree (structure, verbatim facts, overview, anchors, per-section hashes) is derived deterministically in code, and that seam is what lets the model step run anywhere a model is available, including inside your editor.
