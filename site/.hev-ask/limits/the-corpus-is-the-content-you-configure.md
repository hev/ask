hev ask searches only the markdown you point it at (an Astro content collection, a Docusaurus docs tree, a set of globs) with no crawler, no sitemap ingestion, and no way to index pages outside the configured corpus such as a hand-written page or a bare route; if a doc is not in the configured corpus it will not appear in search, so put what you want searchable where the adapter looks.
