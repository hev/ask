Verify the install three ways: keyword search where typing a single heading word deep-links the top result to that section, agentic search where a multi-word question or a suggested one streams a grounded answer with the model's sub-queries shown and inline deep links, and anchors where the verify command exits non-zero if any chunk anchor is missing from the built HTML so it should be wired into CI.
