Prerequisites are Astro 5 with at least one content collection, a server or hybrid adapter since the endpoint route renders on demand and a fully static build cannot serve it, and a provider API key to enable agentic search; keyword search needs no key, and OpenAI and OpenRouter work too via the provider option though the guide uses the Anthropic default.
