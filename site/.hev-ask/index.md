hev ask distils a docs site into a compact directory with one small markdown file per section, a host-neutral mirror built from your markdown rather than your renderer that an agent navigates like any directory and a keyboard-opened overlay answers your readers from. It ships a turnkey Astro integration while every other framework adds the same overlay with one script, you build the digest offline with the bundled skill and commit it, and it suits technical docs, internal wikis, and other medium-sized corpora.
