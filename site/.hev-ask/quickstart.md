The quick start adds hev ask to an existing Astro 5 docs site whose content lives in a collection in about five minutes: you get instant keyword search working first with no key and nothing to host, then add a provider key on the server so the same overlay also runs the agentic answer loop on Enter, or skip the key for the search UI only. It is the Astro path; on any other framework you drop in the same overlay as a tag and point it at a hosted endpoint for answers.
