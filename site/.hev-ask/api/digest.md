The digest is a committed directory built offline and globbed into the build through a virtual module so the running site reads it without filesystem access. It is the agent's source-grounded, progressively-read mirror of your docs: humans read the rendered pages while an agent reads the tree, and every section carries a URL and anchor back to its source. This site renders its own digest for inspection.
