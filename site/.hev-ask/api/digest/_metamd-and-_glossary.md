The meta file carries digest-level fields in frontmatter (version, generation time, content hash as the freshness gate plus ordering) and, in its body, the orientation context, the deterministic overview map, and the suggestions shown on open. The glossary holds one file per term with its aliases and definition, widening keyword search because each query term picks up its aliases before retrieval.
