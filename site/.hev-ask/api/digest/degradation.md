The tree is read defensively: if it is missing or malformed the integration uses what it can, with the agentic loop falling back to keyword-style search, ranking falling back to raw token overlap, and the overlay showing no suggested questions, never hard-failing. Because it is committed markdown, each section's distilled prose and grounded facts change together in one reviewable diff, which is why it is a directory rather than a runtime computation and why the model step can move into a skill.
