Explains how the fields drive behavior: the title-tree and summaries are injected into the loop's prompt-cached system prompt so the loop opens needed sections and quotes facts verbatim; the glossary drives query expansion; terms, summary, and facts lift relevant sections in keyword ranking (falling back to raw token overlap with no tree); the anchor is what verify checks and every citation links to; and the hash is the incremental gate.
