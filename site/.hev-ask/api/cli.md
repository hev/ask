The ask binary has two command groups: keyless read verbs that treat the committed digest as a directory (map it, read a node, pull its grounded literals, search it) plus a synthesized cited reply and an MCP server, and producer commands under a digest subgroup that build, emit, assemble, verify, report on, and migrate the tree.
