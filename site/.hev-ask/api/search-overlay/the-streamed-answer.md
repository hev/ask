Pressing Enter with a key configured replaces the keyword rows with an answer panel: the model's sub-queries appear live as a faint activity line, then the grounded answer streams in token-by-token with a blinking caret. Inline deep links in the prose are styled in the accent color and point at the exact section, a sources row beneath lists every section drawn from, and links are validated against the streamed source set so any link outside it renders as plain text, preventing a hallucinated anchor from becoming a clickable dead link.
