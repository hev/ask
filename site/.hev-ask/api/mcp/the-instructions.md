The server's instructions string is half the product, teaching the agent two things: navigation (read the inline title-tree first, read a section only when its title says it is relevant, search for specifics, lean on the glossary, and do not read every file) and citation (answer from these files and cite every claim with the section's URL and anchor as a deep link). Citation is the part easily lost once an agent synthesizes off local files and is the entire grounded-deep-link value, so it is made non-negotiable.
