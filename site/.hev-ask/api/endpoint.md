The integration injects one on-demand route whose base path serves the overlay (keyword mode returns JSON, agentic mode streams a grounded answer as Server-Sent Events) and whose keyless sub-routes expose the committed digest for CLIs, MCP servers, and generated clients. An OpenAPI contract is published alongside it.
