The overlay component renders the keyboard-opened command palette and is added once in a global layout; it opens over the page as an element that does not affect layout until opened. Its description spans props, the opener attribute, the keyboard and mode model, CSS-variable theming, and the script-tag drop-in for Docusaurus, VitePress, MkDocs, or any static site.
